Association d'Acuponcture A

Notre association

AAA est une association Loi de 1901 créer pour les férues de l'acuponcture. Le but de l'acossiation et de ce site est de regrouper une base de connaissances commune sur l'accuponcture et ainsi permettre de diagnostiquer toutes les maladies de manière précise et juste .
